Le fichier Munchkin.jar doit se trouver à la racine du projet. Pour l'exécuter, il faut aller dans le dossier du projet avec un cmd et écrire :
java -jar Munchkin.jar --module-path "PATH_TO_FX" --add-modules=javafx.controls,javafx.fxml
/!\ Il faut remplacer "PATH_TO_FX" vers votre chemin pour aller vers le dossier lib de JavaFX.
